## 🌟 Overview
**Kumpoo Badminton Shop** is a premium, high-performance e-commerce platform dedicated to authentic Japanese Kumpoo badminton gear. This version is a complete **2026 Modernization** overhaul, focusing on deep interactivity, sleek aesthetics, and advanced frontend logic.

> [!IMPORTANT]
> This project follows a clean **Assets Architecture** and is fully optimized for **Dark Mode** and **Bilingual (EN/VI)** users.

---

## 🚀 Key Features

### 🤖 Smart Chatbot Assistant
Integrated an automated AI-like chatbot with a custom UI. 
- **Bilingual Response:** Automatically switches logic between EN/VI based on site settings.
- **Keyword Recognition:** Instantly handles queries about warranty, product suggestions, and pricing.
- **Typing Simulation:** Mock typing indicator for a more human-like interaction.

### 🔍 Real-time Smart Search
A powerful frontend search engine that filters through the entire product catalog (`productsData`) instantly.
- **Fuzzy Search:** Matches by product name, category, or English translations.
- **Live Filtering:** Dynamic product grid updates without page reloads.

### 🌓 Professional Dark Mode & Glassmorphism
- Smooth toggle transition between Light and Dark themes.
- **Glassmorphism UI:** Translucent headers and modals for a contemporary premium feel.

### 🔄 Dynamic UI Elements
- **Review Slider:** Auto-playing testimonial section with professional navigation controls (Dots & Arrows).
- **Promo Banner:** High-conversion animated hero section with progress bar.

### 🛒 Advanced Shopping Features
- **Wishlist System:** Save your favorite items locally (LocalStorage). Persistent badge counter in the header.
- **Cart Management:** Full cart logic with quantity controls and local storage persistence.
- **Multilingual Support:** One-click seamless toggle between English & Vietnamese.
